Задача 6
--------

Чтобы задать количество виртуальных ядер и памяти в виртуалке,  
прописываем в файле **Vagrantfile** секцию вида
  
 config.vm.provider "virtualbox" do |v|  
    v.memory = 1024  
    v.cpus = 2  
 end
